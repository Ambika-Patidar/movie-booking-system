# Movie Booking System

## Project setup

  Ruby version: 2.4.3
  Rails version: 5.2.5

1. Install project dependencies
  
```sh
rake db:create 
```

2. Create database

```sh
  rails db:create
```

3. Migrate the database

```sh
  rails db:migrate
```

4. Populate the database with seed :

Initial data for admin, movies, screens, screen_layouts, etc.

```sh
rake db:seed
```
